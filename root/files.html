Atlas WebServer File Console

文件管理台

这个页面直接调用当前 C++ 服务端的登录、鉴权、上传、列表、下载和操作日志接口，用来展示完整的小文件业务闭环，而不是零散的演示页面。

上传与状态

复用当前登录态上传文件并查看结果，不需要在这个页面再次登录。

请先从登录后的欢迎页进入文件管理台。
当前演示仅支持 64 KB 以内的小文件上传，前后端限制保持一致。
等待操作。

文件与操作日志

支持刷新文件列表、下载、删除，并查看最近操作日志。

文件列表

所有数据都来自 `/api/private/files` 和 `/api/private/operations`。
登录后可以查看文件列表。

操作日志

用于展示登录、上传、下载、删除等操作行为。
登录后可以查看操作日志。
