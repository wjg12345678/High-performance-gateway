Root 页面规范
===============

当前规范后的主页面：

* `index.html`：统一入口页
* `login.html`：登录页
* `register.html`：注册页
* `login-error.html`：登录失败页
* `register-error.html`：注册失败页
* `welcome.html`：登录后欢迎页
* `files.html`：文件管理台

兼容别名页面：

* `judge.html` -> `index.html`
* `log.html` -> `login.html`
* `logError.html` -> `login-error.html`
* `registerError.html` -> `register-error.html`

历史 action 标志位：

* `0` 注册
* `1` 登录
* `2` 登录检测
* `3` 注册检测
* `5` 历史图片页
* `6` 历史视频页
* `7` 历史关注页
